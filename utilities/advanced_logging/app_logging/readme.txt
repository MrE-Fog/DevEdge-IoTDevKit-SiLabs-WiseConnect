Description  of "app_logging" folder
===========================================================================================================
   Below files are required for the sapi/app logging enhancement support,
   
   advanced_logs_generator.py : Pre-processing script to generate the sapi/app logs.
   manifest.json : JSON file to load inputs required for decoding the sapi/app logs.
   sapi_log_decoder.py : Script to decode the sapi/app logs.

   More details can be found at the link below,
   https://docs.silabs.com/rs9116-wiseconnect/latest/wifibt-wc-sapi-reference/logging-user-guide
